Print Pandas Commands

This option will create a pdf file of the currently-selected command set, as displayed on the Home screen. To amend the selected set of commands, return to the Home screen and adjust the filter options before printing.

Enter a filename for the generated pdf file, and click the button to either print a condensed list of commands (as per the Home screen), or an expanded list with a full description of each command. A footer can also be added to each page if required - this will have the current date and time at the left-hand margin, and the page number at the right-hand margin.

The pdf file will be downloaded to the default download folder, and also displayed on the screen for checking (this behaviour may be different, as it is determined by the browser being used and the browser options that have been set).
